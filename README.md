# Self-Made Grame Framework

C++와 SDL2로 직접 만든 **자작 게임 프레임워크**입니다.  

[2D 게임](https://github.com/Woo95/SDL2_Italian_Brainrot_Survivors) 제작에 필요한 **기본 구조와 시스템**을 제공하며, **유지보수와 확장성**을 고려하여 설계되었습니다.

자세한 개발 및 설계 내용은 [개발 노트](https://woo95.github.io/Devhub/Korean/02_%EA%B2%8C%EC%9E%84-%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8/%EC%9E%90%EC%9E%91-%EA%B2%8C%EC%9E%84-%ED%94%84%EB%A0%88%EC%9E%84%EC%9B%8C%ED%81%AC/)에서 확인할 수 있습니다.
